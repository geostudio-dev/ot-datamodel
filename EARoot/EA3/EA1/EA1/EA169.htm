The MineralOccurrence package explicitly models the properties of those features commonly stored in mineral resource databases.  These are economic or sub-economic concentrations of commodities that may, but need not, have had mining activity associated with them. <br/><br/>The MineralOccurrence package key points are:<br/>1) Describes Earth Resources independent of associated human activities (i.e. mining); <br/>2) Caters for description of Earth Resource using mineral deposit models that describe the actual deposit type; mineral systems that describe the processes associated with deposit formation; and supergene processes; <br/>3) Utilises GeoSciML Mapped Feature to describe spatial representation and Earth Material to describe host, raw and associated materials; <br/>4) Provides the ability to describe commodity resources formally or informally.<br/>
