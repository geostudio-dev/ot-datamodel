A Borehole Collar is a feature corresponding to the start point of a borehole which are often plotted on a map.  <br/><br/>If a text descripiton of the location is available, it should be placed in the gml:description for that feature.  If no GM_Point is available, an OGC nil value (eg; <a href="file://http://www.opengis.net/def/nil/OGC/0/inapplicable"><font color="#0000ff"><u>http://www.opengis.net/def/nil/OGC/0/unknown)</u></font></a><br/>should be used.<br/><br/>Implementers delivering 3-D collar locations should provide and elevation to improve interoperability.<br/>
