Attribute Scope Type
currentLocation Public Location
sequenceNumber=7
Notes: If present, the attribute <i>currentLocation:Location </i>shall describe the location of a physical specimen. This may be a storage location, such as a shelf in a warehouse or a drawer in a museum. <br/>NOTE If a specimen no longer exists, for example it was destroyed in connection with an observation act, then the currentLocation should be omitted or carry a suitable null indicator. <br/><br/>
materialClass Public GenericName
sequenceNumber=1
Notes: The attribute <i>materialClass:GenericName </i>shall provide a basic classification of the material type of the specimen. <br/>EXAMPLE soil, water, rock, aqueous, liquid, tissue, vegetation, food. <br/><br/>
samplingLocation Public GM_Object
sequenceNumber=4
Notes: If present, the attribute <i>samplingLocation:GM_Object </i>shall describe the location from where the specimen was obtained. <br/>NOTE Where a specimen has a relatedSamplingFeature whose location provides an unambiguous location then this attribute is not required. However, if the specific sampling location within the sampledFeature is important, then this attribute supports its description. <br/><br/>
samplingMethod Public SF_Process
sequenceNumber=3
Notes: If present, the attribute <i>samplingMethod:SF_Process </i>shall describe the method used to obtain the specimen from its sampledFeature. <br/><br/>
samplingTime Public TM_Object
sequenceNumber=2
Notes: The attribute <i>samplingTime:TM_Object </i>shall record when the specimen was retrieved from the sampled feature.<br/><br/>
size Public Measure
sequenceNumber=6
Notes: If present, the attribute <i>size:Measure </i>shall describe a physical extent of the specimen. This may be length, mass, volume, etc as appropriate for the specimen instance and its material class. <br/><br/>
specimenType Public GenericName
sequenceNumber=8
Notes: If present, the attribute <i>specimenType:GenericName </i>shall describe the basic form of the specimen. <br/>EXAMPLE polished section; core; pulp; solution. <br/><br/>

One or more procedures may have been applied to a specimen. May contain collection, sampling and preparation procedures
Details:
In many applications specimen preparation procedures are applied to the material prior to its use in an observation. The association <i>Preparation </i>shall link a SP_Specimen to a OM_Process that describes a phase of the specimen preparation.  The association-class SP_PreparationStep provides semantics for the association. <br/><br/>
