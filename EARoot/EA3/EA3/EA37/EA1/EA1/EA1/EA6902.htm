The class <i>NamedValue</i> provides for a generic soft-typed parameter value. NamedValue shall support two attributes. <br/><br/><br/>
  • Attributes
  • Tagged Values
  • Advanced
Attribute Scope Type
name Public GenericName
sequenceNumber=1
Notes: The attribute name:GenericName shall indicate the meaning of the named value. Its value should be taken from a well-governed source if possible. <br/><br/>EXAMPLES When used as the value of an Observation:;parameter, the name might take values like ‘procedureOperator’, ‘detectionLimit’, ‘amplifierGain’, ‘samplingDepth’. <br/><br/>
value Public Any
sequenceNumber=2
Notes: The attribute value:Any shall provide the value. The type Any should be substituted by a suitable concrete type, such as CI_ResponsibleParty or Measure. <br/><br/>
