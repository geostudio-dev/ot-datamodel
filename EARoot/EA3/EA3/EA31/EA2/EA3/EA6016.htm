The LC_PeriodicVariation component is related to the LC_LC_WaterBodyAndAssociatedSurfaceElement metalanguage object by the relationship <i>periodicVariation</i>. It has four optional attribute, <i>periodType, persistancePeriod, peristanceUnits </i>and<i> description</i>. The attribute <i>periodType</i> describes the type of variation  by use of the code list LC_PeriodVariationType. The attribute <i>persistancePeriod</i> describes the persistence period as a time duration. A constaint exists requiring that persistencePeriod always be greater than 0. The attribute <i>persistanceUnits</i> describes the time units used for the persistence period. A constaint exists requiring that persistence units be used if and only if a persistence period exists. The attribute <i>description</i> provides an optional textual characterization of the  persistence types. A constaint exists requiring that a description exist only if a persistence type from the code list has been chosen.. The component LC_PeriodicVariation may be used to refine the metalanguage object LC_WaterBodyAndAssociatedSurfaceElement to permit the generation of a more specific land cover feature class in a land cover classification system.<br/>
