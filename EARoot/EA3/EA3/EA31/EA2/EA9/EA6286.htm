EL_Agroforestry:LC_Stratum has the attribute <i>presenceType</i> = Fixed, meaning that the element is not a conditional relationship and the attribute <i>onTop</i> = Baseline, meaning that the element is with respect to the base level of the stratum, not on top of any other stratum. This is not shown in the model because it is the default value for the attributes. <br/><br/>The attribute <i>name </i>= tree.<br/><br/>
