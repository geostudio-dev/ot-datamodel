The LC_PermittedPosRealValue metalanguage object is a subtype of LC_PermittedRealValueType. It is a metalanguage object for a positive real number value permitted in the metalanguage where the attribute value may be instantiated as a real number type using the Real type from the basic Primitive Types package in ISO/TS 19103, with the constraint: {Value #gt;= 0.0}.<br/>
