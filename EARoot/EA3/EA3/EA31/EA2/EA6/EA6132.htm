The LC_ValueClass is a metaclass that describes how permitted numeric values at the meta level maybe instantiated to the basic number types form ISO 19103 at the type level. The LC_ValueClass metaclass is a supertype of the three value metaclasses LC_PermittedRealValueType, LC_PermittedPercentageValueType and LC_PermittedIntegerValueType. Each of these value classes has subtypes that describe a number and a range. Each of the value classes is described by a constraint that characterises the type of numerical entity that may be represented by the value class. This is represented in Figure 25.<br/>These value classes are generators for the actual value objects that will appear in instances of a classification system generated using the LCML expressed as a UML metamodel. The result of instantiating these classes are also value classes taken from the basic types provided in ISO 19103. The results of instantiating a classification system are objects that carry actual values. In the various examples given in this international standard value classes have been instantiated two levels from the meta level to the type level and then to the instance level, to produce real objects as needed in an example.<br/>
