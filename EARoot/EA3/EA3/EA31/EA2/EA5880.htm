The LC_LandCoverClassStructure package contains the high level structure of the LCML.<br/><br/>The LCML is a metamodel which may be used to describe a wide variety of LCCS. The LCML operates by describing each class in an LCCS in terms of a set of basic elements that when combined describe each aspect of the LCCS class. That is, each class in an LCCS may be modelled using the basic element metaobjects defined in the LCML. These elements are all subtypes of the metaobject LC_Element, so any particular LCCS class can be described as a combination of a set of LC_Element subtypes.<br/><br/>
